## üöÄ Featured Project: Trafet
**Trafet** is a high-performance **Map & Travel** assistant. It demonstrates a robust implementation of **Compose Multiplatform (CMP)** with a strong focus on shared business logic and reactive UI.

- üß™ **Tech Stack**: Kotlin / Compose Multiplatform (KMP), Node.js Cloud Functions, and Firestore.
- üèóÔ∏è **Architecture**: Strictly follows **Clean Architecture** (Domain, Data, and UI layers) with **ScreenModel** for state management.
- üö¶ **State Management**: Reactive UI powered by **StateFlow** and **SharedFlow**, ensuring a seamless 60fps experience.
- üì° **Data Flow**: Implements **DTO to Domain model** mapping, fetching data from **Cloud Functions** (Node.js) which integrates with **Firestore** and **Google Routes API**.
- ‚öôÔ∏è **Advanced Backend**: Utilizes **Google Cloud Tasks** for automated Firestore updates and triggered **Push Notifications**, ensuring reliable background processing.
- üì¶ **Offline Capabilities**: Supports offline route and map viewing by caching critical data using **Kstore**.
- üí≥ **Monetization**: Production-ready In-App Purchase (IAP) integration powered by **RevenueCat**.
- üõ†Ô∏è **Tools**: Android Studio, Xcode, VSCode, Figma, Git, SourceTree
